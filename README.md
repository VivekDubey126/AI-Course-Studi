## 🧠 Overview

**AI Course Studio** is a full-stack web application that leverages **Google Gemini AI** to instantly generate professional, university-grade course curriculums on any topic. Users enter a topic, and the platform produces a complete learning path — including structured syllabi (8–10 modules), dual-perspective YouTube video tutorials, rich markdown notes, adaptive quizzes with AI-generated explanations, and curated external resources. An integrated **agentic AI chatbot** provides real-time learning assistance with multi-turn conversational memory.

---

## ✨ Key Features

### 🎓 AI-Powered Course Generation
- Enter any topic and receive a **structured syllabus with 8–10 specialized modules** generated by Google Gemini 2.5 Flash
- Each chapter includes **rich markdown notes**, **5 adaptive quiz questions with AI explanations**, and **curated external learning resources**
- Courses are persisted in MongoDB — revisit anytime without re-generation

### 📹 Dual-Perspective Video Learning
- Integrated **YouTube Data API v3** to auto-curate **two tutorial videos per chapter** from top educators
- Unique **"Perspective A / Perspective B"** toggle lets learners compare explanations from different instructors
- Automatic fallback and replacement search for unavailable videos

### 💬 Agentic AI Chatbot
- Built-in **AI learning assistant** powered by Google Gemini with system-level prompt engineering
- Maintains **full conversation history** for contextual multi-turn interactions
- Supports concept explanations, code debugging, learning path suggestions, and course recommendations
- Markdown-formatted responses with code highlighting

### 🔐 Dual Authentication System
- **Email/Password registration** with bcrypt password hashing (salt rounds: 10)
- **Google OAuth 2.0** single sign-on via Passport.js
- **JWT token-based** session management with auto-login from persisted tokens
- Protected API endpoints via custom auth middleware

### 📊 Interactive Dashboard
- **Full-text search** across course titles and descriptions
- **6 sorting criteria**: Newest, Oldest, Most Progress, Least Progress, A→Z, Z→A
- **4 progress filter states**: All, Not Started, In Progress, Completed
- **Soft-delete** (hide) functionality for course management
- Real-time animated state transitions and result count display

### 🎬 Immersive Course Player
- **Resizable split-pane layout** — drag to adjust video vs. content height
- Tabbed content viewer: **Notes** (rendered markdown) | **Resources** (linked cards) | **Quiz** (interactive with instant feedback)
- **Chapter-level progress tracking** with visual completion indicators
- Chapter sidebar with completion status and navigation

### 🎨 Premium UI/UX
- **Dark-mode glassmorphism** design with backdrop blur and gradient effects
- **Framer Motion** animations — page transitions, layout animations, spring physics
- **GSAP + ScrollTrigger** — scroll-triggered hero animations and staggered card reveals
- Infinite-scroll **review carousel** with edge fade gradients
- Responsive design optimized for desktop and mobile

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- MongoDB Atlas account (or local MongoDB)
- Google Gemini API Key
- YouTube Data API v3 Key
- Google OAuth 2.0 credentials (Client ID + Secret)

### Installation

```bash
# Clone the repository
git clone https://github.com/arnab5565/AI-Course-Studio_beta.git
cd AI-Course-Studio_beta

# Install server dependencies
cd server
npm install

# Install client dependencies
cd ../client
npm install
```

### Running the Application

```bash
# Terminal 1 — Start the backend server
cd server
npm run dev          # Runs on http://localhost:5000

# Terminal 2 — Start the frontend dev server
cd client
npm run dev          # Runs on http://localhost:5173
```

## 📡 API Endpoints

### Authentication
| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/api/auth/register` | Register with email/password |
| `POST` | `/api/auth/login` | Login with email/password |
| `GET` | `/auth/google` | Initiate Google OAuth flow |
| `GET` | `/auth/google/callback` | Google OAuth callback |

### AI Generation
| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/api/gemini/syllabus` | Generate course syllabus from topic |
| `POST` | `/api/gemini/chapter` | Generate chapter content + YouTube videos |

### Course Management (Protected — requires JWT)
| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/api/courses/enroll` | Save & enroll in a generated course |
| `GET` | `/api/courses/my` | Fetch user's enrolled courses |
| `PATCH` | `/api/courses/:id/hide` | Soft-delete (hide) a course |
| `PATCH` | `/api/courses/:id/chapter/:chapterId/progress` | Mark chapter as completed |
| `PATCH` | `/api/courses/:id/chapter/:chapterId/content` | Save AI-generated chapter content |

### Chatbot
| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/api/chatbot/chat` | Send message with conversation history |

## 📝 Resume Key Points

### Architecture & Full-Stack Development
- Designed and developed a **full-stack web application** using **React 19 (TypeScript) + Vite** on the frontend and **Node.js/Express 5** with **MongoDB (Mongoose)** on the backend, following an MVC architecture pattern.
- Architected a **RESTful API** with dedicated routes for authentication, AI content generation, course management, and chatbot interactions, protected by **JWT-based middleware**.

### AI / Generative AI Integration
- Integrated **Google Gemini 2.5 Flash API** with structured JSON schema responses to dynamically generate course syllabi (8–10 modules), chapter-level educational content, quiz questions with explanations, and curated external learning resources.
- Built an **agentic AI chatbot** powered by Gemini with multi-turn conversational memory, system-level prompt engineering, and markdown-formatted responses for real-time learning assistance.

### Third-Party API Integration
- Implemented **YouTube Data API v3** integration to auto-curate dual-perspective tutorial videos per chapter, enabling a unique "Perspective A / Perspective B" learning methodology.

### Authentication & Security
- Implemented a **dual authentication system** supporting both traditional email/password (bcrypt-hashed) registration and **Google OAuth 2.0** (via Passport.js), with **JWT token management** and auto-login from persisted tokens.

### Data Persistence & Course Management
- Designed a **MongoDB data model** with nested schemas (Course → Chapters → Quizzes/Resources/Videos) to persist AI-generated content, enabling users to resume courses without re-generation.
- Developed course **enrollment, progress tracking (chapter-level completion), and soft-delete** functionality with real-time UI updates.

### Frontend & UI/UX
- Built a premium, **dark-mode glassmorphism UI** with **Framer Motion** animations, GSAP scroll-triggered effects, resizable split-pane course player, and an infinite-scroll review carousel.
- Implemented a feature-rich **dashboard** with full-text search, multi-criteria sorting (6 options), progress-based filtering, and animated state transitions.
